**問題** **関数**を定義する方法と関数を呼び出す方法を確認せよ．

引数を一つ持つ関数．

```python
#関数定義
def f(x):
  return x * 2
```

関数を呼び出して，**戻り値**（返り値，返値）を得る．

```python
#
f(3)
```
